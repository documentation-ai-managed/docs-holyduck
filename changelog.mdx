## Upgrading to the Latest Version

Follow these steps to upgrade HolyDuck smoothly. Always back up your data first.

<Callout kind="alert">
Before upgrading, review breaking changes in the release notes above.
</Callout>

<Steps>
  <Step title="Backup Your Data" icon="database">

    Export your projects to avoid data loss.

    ```bash
    holyduck export --project=all --format=json > backup.json
    ```

  </Step>
  <Step title="Update Package" icon="download">

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install holyduck@latest
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn add holyduck@latest
````
      </Tab>
      <Tab title="Docker" icon="docker">
````bash
docker pull holyduck/holyduck:latest
docker run -p 3000:3000 holyduck/holyduck:latest
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Apply database schema updates.

    ```bash
    holyduck migrate up
    ```

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

    Restart to apply changes.

    ```bash
    holyduck restart
    ```

  </Step>
</Steps>
